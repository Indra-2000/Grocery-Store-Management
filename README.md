**Overview**

This project demonstrates the design and implementation of a Grocery Store Management Database using SQL. It includes database creation, table relationships, constraints, and SQL queries to manage and analyze grocery store operations efficiently.

**Key Features**
Database design with relational tables and constraints.
Management of products, categories, customers, suppliers, employees, and orders.
SQL queries for sales analysis and business reporting.
Data retrieval using joins, aggregations, subqueries, and filtering techniques.

**Technologies Used**
SQL
MySQL / Relational Database Concepts

**Project Objectives**
Organize and manage grocery store data efficiently.
Track sales, inventory, and customer transactions.
Generate meaningful business insights through SQL queries.

**Skills Demonstrated**
Database Design
SQL Query Writing
Joins & Subqueries
Aggregate Functions
Data Analysis & Reporting
Relational Database Management
